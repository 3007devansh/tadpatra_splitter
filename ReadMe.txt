Version 1.0.0.0 Contains:
- Batch Processing
	* Select Input Folder
	* Output Folder Name will be selected automatically with suffix of _Splitted word.
	e.g. If Input Image Folder is: D:\Images
			Output Image Folder default name will be D:\Image_Splitted
			
	* Select appropriate value of 'Output Border Pixel Margin'.
	* Select 'First Folio is Title Image' if the first image is of index path (e.g. Bhandar Title Page)
	* Click on 'Start Splitting'.
	* Splitting will be done on the selected folder and splitted images will be stored in 'Output Image Folder'.

- Manual Processing
	* Manual processing can be done on two images, e.g. Front and Back side images
	* Select both the files manually
	* Write 'Output ManuScript Number'
	* Select appropriate value of 'Output Border Pixel Margin'.
	* Click on 'Start Splitting'.
	* Splitted images of 'Front' and 'Back' both will be stored in 'Output Image Folder'.
	
- Image Angle Correction
	* Select 'Input Image Folder' containing the images where you want to skew / rotate angle.
	* First image in that folder will be automatically shown in the Grid.
	* Use <- (Left) arrow key to navigate to the previous image.
	* Use -> (Right) arrow key to navigate to the next image.
	* Select appropriate angle for AntiClockWise rotation. Default value is 1. Use Ctrl + Up arrow key to do AntiClockWise rotation.
	* Select appropriate angle for ClockWise rotation. Default value is 1. Use Ctrl + Down arrow key to do AntiClockWise rotation.
